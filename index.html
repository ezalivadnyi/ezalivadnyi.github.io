How you can create website like this, with github.io domain name?

With help of this service: pages.github.com. You can easily create a static web-page like this. All you must to do is just crate github repository with name "yourlogin.github.io". Вы можете сделать себе такой же сайт, просто создав на GitHub репозиторий с именем вашлогин.github.io и поместив в него index.html файл. Если у вас нет аккаунта, то вам придётся зарегистрироваться на гитхабе.
После этого по адресу http://вашлогин.github.io будут доступны все файлы через стандартный порт 80.
У вас появится гит - репозиторий, который вы сможете редактировать в своей IDE.
Чтобы обновить файлы на таком сайте - достаточно сделать Commit и Push.
